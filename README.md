# docs-server

Содержит сервер документаций микросервисов.

## Запуск приложения

### Сетевые порты

| Номер | Описание             |
|-------|----------------------|
| 8080  | Порт основного API.  |
| 9090  | Порт служебного API. |

### Существующие переменные окружения

| Имя                       | Описание                                         | Значение по умолчанию |
|---------------------------|--------------------------------------------------|-----------------------|
| HTTP_BASE_PATH            | Базовый путь основного API.                      | /v1/docs-server       |
| HTTP_MAX_HEADER_SIZE      | Максимально допустимый размер заголовка запроса. | 8KB                   |
| HTTP_MANAGEMENT_BASE_PATH | Базовый путь служебного API.                     | /v1/docs-server       |
| CONFIG_SERVER_URL         | Ссылка на сервер конфигураций.                   |                       |

## Работа с приложением

### Чтение документации API

Документация API должна быть доступна по адресу: https://api.sogeor.com/v1/docs-server/swagger-ui
